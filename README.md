# ジグソーパズルゲーム

ブラウザで動作するジグソーパズルゲームです。ローカルの画像ファイルを選択してパズルを楽しめます。

## 機能

- ローカル画像ファイルからパズルを生成
- 5段階の難易度（12〜300ピース）
- 本格的なジグソーピース形状（凸凹のあるピース）
- ピース同士の自動スナップ接続
- 接続されたピースはグループとして一緒に移動
- ヒント表示機能
- タイマー・進捗表示
- レスポンシブデザイン（モバイル対応）

## 使い方

1. `index.html` をブラウザで開く
2. 「画像を選択」から好きな画像ファイルを選ぶ
3. 難易度を選択（ピース数が自動計算されます）
4. 「ゲーム開始」をクリック
5. ピースをドラッグして正しい位置に配置
6. 近くに正しいピースがあると自動的にスナップして接続

## ファイル構成

```
pazzle/
├── index.html   # メインHTML
├── style.css    # スタイルシート
├── script.js    # ゲームロジック
└── README.md    # このファイル
```

## 技術仕様

- HTML5 Canvas でピースを描画
- ベジェ曲線でジグソーピースの凸凹形状を生成
- タッチイベント対応（スマートフォン・タブレット）
- 外部ライブラリ不使用（バニラJS）
